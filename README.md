## 📌 Featured Projects

### [Laravel Tasks API](https://github.com/mohamad-aka-slim/laravel-tasks-api)

A RESTful task management API built with Laravel, featuring authentication, task CRUD, filtering, pagination, caching, and API-focused structure.

### [Simple Database PHP Class](https://github.com/mohamad-aka-slim/simple-database-php-class)

A minimal async ICMP ping tool written in Rust using Tokio and surge-ping.
